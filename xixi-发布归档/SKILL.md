# 溪溪·发布归档

## 第六步：LLM 智能全局优化

**输入**：成品视频/音频 + 文案 + 分镜JSON
**输出**：`06_优化报告.md`

**优化维度**（详见 `../references/06-LLM智能优化.md`）：

1. **科学准确性**：知识点是否正确、表述儿童能理解
2. **文案润色**：笑点密度、对话够"人话"
3. **儿童安全**：危险模仿风险、不适当内容
4. **节奏调整**：儿童注意力~15秒一个刺激点
5. **家长友好度**：家长是否愿意让孩子看

**操作**：传入成品+文案+分镜 → 多维度审查 → 输出优化报告 → 执行修改

## 第八步：发布

> 详见 `../references/08-发布.md`

**核心产出**：`发布包/发布信息.md`

### 标题方案（3选1）

| 风格 | 模板 | 示例 |
|------|------|------|
| 🔥 悬念型 | `[知识点]居然[反常识]？[时长]看完就懂` | 放屁会臭居然是因为臭鸡蛋？45秒看完就懂 |
| 😂 翻车型 | `[神仙]做[实验]翻车了！闹闹：[吐槽]` | 哪吒用风火轮加热试管又炸了！闹闹：第七次了 |
| 📖 知识型 | `儿童科学冷知识：[一句话知识点]` | 儿童科学冷知识：你的屁里有硫化氢 |

**推荐标题**：从3个方案中选最佳。

### 标签

| 平台 | 标签 |
|------|------|
| 抖音 | `#儿童科普 #天庭幼儿园 #仙问百答 #亲子教育 #[知识点标签]` |
| 喜马拉雅 | `#有声书 #AI配音 #儿童科普 #天庭幼儿园 #仙问百答` |

### 发布检查清单

```
- [ ] 音频试听/视频预览无断句角色错位
- [ ] 音量正常(-16 LUFS)，无爆音
- [ ] 片头0.5s淡入，片尾1s淡出
- [ ] 角色声线无串音
- [ ] 标题含角色名+知识点
- [ ] 科学知识点核查无误
- [ ] 儿童安全审查通过
- [ ] 家庭小实验材料安全
```

### 上传

| 平台 | 方式 |
|------|------|
| 🎬 抖音 | `creator.douyin.com` → 上传MP4 + 填写标题/标签 |
| 🎧 喜马拉雅 | `python ../scripts/ximalaya_upload.py 发布包/` 或手动 `studio.ximalaya.com` |
