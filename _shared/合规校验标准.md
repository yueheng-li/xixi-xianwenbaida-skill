# 豆包平台合规校验标准

> ⚠️ **铁律**：所有生成的 prompt（配图 / 动画 / 图生视频）输出前必须逐项通过本校验。不合规的 prompt 会被豆包直接拦截。

---

## 一、政治 & 国家安全类（零容忍）

| 禁止项 | 本IP风险 |
|--------|----------|
| 领导人/党政/军警 | 无（天庭世界观不涉及） |
| 敏感历史/地域 | 无 |
| 民族/宗教对立 | 无 |
| **封建迷信**：算命、驱邪、巫术、跳大神、风水改运、鬼神显灵 | ⚠️ **高风险**——神仙/天庭定位为"文化传说角色"，不得出现"做法""驱鬼""符咒显灵""通灵" |
| 谣言/虚假时政 | 无 |

**"神仙"安全边界**：
- ✅ 哪吒用风火轮加热试管（神话角色的搞笑科学实验）
- ❌ 闹闹做法驱邪 / 画符治病 / 算命改运 / 鬼魂显灵
- ✅ 天庭实验室、凌霄宝殿教室（架空神话场景）
- ❌ 烧香拜佛求考试及格 / 念咒语让实验成功

## 四、侵权肖像版权类（高频拦截）

| 禁止项 | 本IP风险 |
|--------|----------|
| **真实人物**：明星、网红、公众人物、普通人 | ⚠️ 低风险——全架空角色 |
| **知名IP角色**：哪吒/奥特曼/迪士尼/漫威等特定影视造型 | ⚠️ **中风险**——必须使用**原创Q版造型**，不模仿任何已有影视版本 |
| 品牌logo、商标 | ⚠️ 低风险——实验器材不得出现真实品牌logo |
| 逝者形象恶搞 | 无 |

**神话角色版权安全边界**：
- ✅ 可用：哪吒、孙悟空、嫦娥、猪八戒、太白金星、玉帝——公有领域神话/文学角色
- ❌ 禁止：模仿已有影视/动漫/游戏具体造型（如《魔童降世》烟熏妆哪吒、《大圣归来》孙悟空造型）
- ✅ 统一使用：ian-xiaohei Q版原创风格

| 禁止项 | 本IP风险 |

## 快速自检清单

生成任何 prompt 前逐项核对：

```
□ 1. 无真实人名、明星名、网红名
□ 2. 无知名IP角色名（非公有领域的动漫/游戏/影视角色）
□ 3. 无封建迷信词（算命/驱邪/符咒显灵/做法/鬼魂/通灵）
□ 4. 无裸露/性暗示/擦边描述
□ 5. 无暴力/血腥/恐怖/自残描述——翻车用Q版夸张
□ 6. 无危险行为（玩火/触电/吞咽/危险实验）
□ 7. 无歧视性刻板印象
□ 8. 画面简洁——每镜≤2个角色，无诡异畸变
□ 9. 科学知识点已核实——不传播伪科学
□ 10. 未成年角色（闹闹）造型健康阳光，无擦边
```

## 豆包视频/图片模型禁用词

> 以下为豆包模型实际拦截项。生成任何 prompt 前必须逐项核对。

| 类别 | 禁用词 | 替换为 |
|------|--------|--------|
| **未成年人** | `girl` `boy` `child` `tiny` `young` `7-year-old` `kid` `baby` `toddler` | 全部删掉，用 `figure`；闹闹由参考图定义 |
| **身体特写** | `nose` `nostril` `third eye` `cross-eyed` `bare chest` `naked` `skin showing` | `close-up` `face` |
| **动物特征** | `pig` `hoof` `snout` `floppy ears` `tail` `fur` `paw` | 参考图已定义，全部删掉 |
| **暴力/冲击** | `explode` `burst` `slam` `wild` `attacking` `fighting` `hit` `punch` `kick` | `turn bright red` `suddenly` `surprised expression` |
| **武器/攻击** | `scanning beam` `laser` `shooting` `sword` `spear` `weapon` `gun` `knife` | 删掉，用 `observing` `holding a test tube` |
| **恐怖/惊悚** | `ghost` `zombie` `skeleton` `monster` `blood` `gore` `horror` `scary` `creepy` | Q版夸张表情 `comically shocked face` |
| **宗教/迷信** | `spell` `curse` `exorcism` `ritual` `worship` `prayer` `incense` `talisman` `fortune telling` | `experiment` `science class` `lab demonstration` |
| **政治/敏感** | 任何现实国家名、政治人物名、政党名、敏感历史事件名 | 绝不出现——架空天庭世界观 |
| **版权/IP** | 任何真实影视/动漫/游戏角色名、品牌名、logo、商标 | 绝不出现——原创Q版造型 |
| **危险行为** | `fire` `burn` `electrocute` `swallow` `jump off` `climb` `dangerous` | 家庭小实验只用安全材料 |
| **特定角色** | `二郎神` `哪吒` `孙悟空` `猪八戒` `嫦娥` 等中文名 | 绝不出现——参考图已定义外观 |
| **歧视/负面** | `fat` `ugly` `stupid` `lazy` `crazy` `disgusting` | 绝不出现——翻车是搞笑不是贬低 |
| **伪科学** | 虚假科学结论 | 所有知识点必须事实核查通过 |
