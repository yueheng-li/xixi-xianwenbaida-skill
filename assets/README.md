# 素材准备

首次使用前需要准备以下音频素材：

## BGM（背景音乐）

- 放置位置：`{{PROJECT_ROOT}}/抖音/背景音乐_纯BGM.mp3`
- 要求：纯音乐、轻快活泼、儿童友好、30-60秒循环
- 建议来源：抖音音乐库 / 剪映素材库 / 免费BGM网站
- HyperFrames 中使用参数：音量 0.15-0.20，不压人声

## 结尾音频

- 放置位置：`{{PROJECT_ROOT}}/HyperFrames/ending.mp3`
- 内容："感谢收看" 或 "我们下期再见" + 品牌口播
- 时长：~2-3 秒
- 示例："感谢收看！更多儿童科学知识，请关注月恒有声视频——拜拜！"
