# 溪溪·文案创作

> **IP**：《天庭幼儿园》—— 7岁科学课代表闹闹 + 翻车神仙。闹闹淡定认真 vs 神仙咋咋呼呼 = 核心笑点。
> **合规**：文案中不得出现封建迷信/危险行为/歧视。神仙=文化传说角色，只做科学实验。

## 第一步：类型判断

| 用户说 | 模式 | 输出 |
|--------|------|------|
| "视频"/"抖音" | 🎬 视频 | 30-50秒，150-300字 |
| "音频"/"有声书"/"月恒" | 🎧 音频 | 10-15分钟，1800-2500字 |
| 未指定 | ❓ | **必须问**："你要视频版还是音频版？" |

## 🎬 视频文案规范

**结构（5段）**：【开场】【核心知识/反转】【翻车高潮】【收尾+实验】【下集预告】

**规则**：
1. 段名用描述性标题如 `【开场】【悟空炸毛】【收尾】`
2. 每句标注角色：`角色："台词"`
3. 旁白用于场景叙事和衔接
4. 开篇：闹闹提科学问题 / 神仙搞砸 / 神仙问蠢问题
5. 中段：闹闹讲科学 + 神仙质疑/抢话/翻车，拟声词丰富
6. 结尾：神仙懂了（或又翻了）+ 家庭小实验彩蛋 + 下一章钩子
7. 单句≤20字，口语化，风趣幽默
8. **首集规则**：第一集旁白必须交代闹闹是谁（"闹闹，七岁，小学科学课代表。太白金星敲了她家的门——天庭缺一个科学老师。"），后续集不重复
9. **下一章钩子**：写文案时必须同时读取下一章正文，提取核心笑点浓缩为1-2句预告。格式："下期预告——太白金星要做大便色卡，玉帝正好来视察……"
10. **家庭小实验**：只用家庭常见材料 + 绝对安全 + 3步以内 + 标注"请在家长陪同下进行"

## 🎧 音频文案规范

**开篇铁律**：
> "哈喽小伙伴们！欢迎收听月恒的有声书。今天聊——xxx。"

**结构**：
```
【开场白 30秒】→ 【引入场景 2-3分钟】→ 【科学讲解 4-6分钟】
→ 【翻车高潮 2-3分钟】→ 【知识总结 1-2分钟】→ 【结束语 30秒】
```

**音频专属要求**：
- 对话体为主，旁白可比视频多（音频需要场景描绘让听众"看见"）
- 拟声词比视频多50%（音频靠声音制造画面感）
- 每3分钟至少一个笑点/翻车/音效刺激
- 闹闹的淡定在音频中更突出——周围炸成一片，她依然慢条斯理

## 审核循环（LOOP）

```
生成文案 → 展示给用户审核 → 等待反馈 →
  ├─ 用户通过("通过"/"OK"/"继续") → 交付文案，流程结束
  └─ 用户不通过(修改意见) → 按意见修改 → 再次展示 → 再次等待
```

**审核规则**：
- 生成后**必须停**，不得自动进入下一步
- 修改只改用户指出的问题，保留认可的部分
- 连续3次不通过 → 主动问："你觉得哪些地方需要调整？节奏、笑点、还是知识深度？"

## 从正文章节提取文案

```
输入：正文/第N章.md
输出：01_文案.md（150-300字视频对话体）
```

- 挑选章节**一个最强笑点**作为视频核心
- 压缩到5段：【开场】【核心知识】【翻车】【收尾+实验】【预告】
- 去掉环境描写/心理活动，只留对话+旁白
- **必须同时读第N+1章**提取下集预告
- 家庭小实验从章节末尾 🔬 提取
