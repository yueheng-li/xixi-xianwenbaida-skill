### 1.2 天庭的基本设定

**天庭是什么**：
- 中国神话体系中的神仙居所和管理机构
- 由玉皇大帝统御，下设各部（雷部/火部/水部/瘟部/斗部等）
- 存在时间：以万年计，远早于人类文明
- 天庭时间流速与凡间不同：天庭一日 ≈ 凡间三日（这个设定可以在后续科普相对论时玩梗）

**天庭在本书中的特殊定位**：
- **职能转变**：传统天庭负责"管理三界"，本书中天庭新增了一个部门——**天庭科学扫盲办公室**（简称"天科办"）
- **核心矛盾**：神仙们掌握着改变物理规律的神力，却完全不懂物理规律本身
- **喜剧来源**：神仙用神力强行解释科学→必然翻车→闹闹用科学给出正确答案

### 1.3 神力与科学的共存法则（本书核心设定）

```
神力的本质：
  神力 ≠ 违反科学
  神力 = 用更高维度的手段实现了科学结果

  例：
  - 哪吒的风火轮能飞 ≠ 万有引力不成立
  - 风火轮能飞 = 风火轮内部产生了一个局部反重力场
    （但哪吒自己不知道，他以为"踩着就能飞"）

  这个设定保证了两件事：
  ① 科学永远是对的（知识准确性底线）
  ② 神力也永远是真的（神仙的体面底线）
  ③ 翻车的原因永远是"神仙不懂科学原理瞎用神力"（笑点引擎）
```

**天科办的成立背景**：
1. 近年来凡间小孩的祈祷内容从"保佑我考上状元"变为"保佑我物理考满分""保佑我科学发明获奖"
2. 天庭客服部收到大量来自小朋友的"为什么"问题，神仙们答不上来
3. 玉帝在凌霄殿展开了一次"科学水平摸底考试"——全体神仙不及格，孙悟空0分（他交了白卷，在上面画了个桃子）
4. 太白金星提出"从凡间请一位科学老师"
5. 哪吒质疑"请个大人来管我们？那我宁可是个小孩。"
6. 于是……闹闹被选中了。

## 三、核心规则系统

### 3.1 闹闹的天庭通行规则

- 闹闹并非随时在天庭——他平时在凡间正常上学（三年级二班）
- 每周三下午（凡间时间），太白金星派一朵"祥云专车"来接他
- 祥云专车有隐身功能，凡间看不到（闹闹妈妈以为是校车）
- 闹闹在天庭的时间流速是独立的：他在天庭待一整天，凡间只过了两分钟
- 手机在天庭可以用（连的是雷公电母维护的"天界WiFi"），但信号时好时坏

### 3.2 神力使用规则（在天科办实验室内）

| 规则 | 内容 | 违规记录 |
|------|------|----------|
| 规则一 | 实验室内禁止用神力作弊（要用就用科学方法） | 哪吒违规7次 |
| 规则二 | 翻车后必须自己打扫（不准用神力一键还原） | 孙悟空违规4次 |
| 规则三 | 做实验前必须预习（闹闹会抽查） | 猪八戒违规无数 |
| 规则四 | 不许用神力欺负实验器材 | 二郎神违规1次（天眼看碎了显微镜的镜头） |
| 规则五 | 实验结果不准用神力篡改 | 哪吒违规3次（把失败的实验用三昧真火烧掉了证据） |

### 3.3 功德与科学水平的兑换

本书不设传统修仙小说的"系统面板"，但有一个隐性的兑换逻辑：

- 神仙做对一道科学题 = 功德+1
- 神仙做错一道科学题但勇于承认 = 功德+2（诚实加分）
- 神仙做错一道科学题但嘴硬狡辩 = 功德-3
- 哪吒因为嘴硬已经被扣成了负功德

## 四、主线暗线

### 4.1 表层主线（搞笑日常）

闹闹教神仙学科学 → 神仙做实验翻车 → 笑点爆炸 → 闹闹总结知识点 → 每章一个家庭小实验

### 4.2 暗线一：天道规则的转变

- **现象**：近一百年来，天庭的某些古老封印出现了松动
- **原因**：凡间科技崛起→人类对世界的理解从"神的意志"变为"物理规律"→信仰之力减弱→封印的底层逻辑（纯神力驱动）开始不稳定
- **解法**：神仙学会科学之后，可以用"神力+科学"双模式维持封印——这是玉帝真正的长远布局
- **揭示节奏**：第1-30章（彩蛋暗示）→第31-60章（疑点出现）→第61-100章（部分揭示）→第101-150章（升级为第二主线）→第151-200章（科学与神力融合，封印以新方式重构）

### 4.3 暗线二：闹闹的身世

- **表象**：闹闹是一个普通的三年级小学生，科学成绩好，被随机选中
- **真相**：闹闹的祖上（曾曾曾祖父）是一位隐居的"凡间散仙"——不是天庭在编神仙，但是有仙缘
- **契约**：每一个甲子年（60年），这个家族的后人中会有一位"与天庭有缘"的孩子，成为连接凡间与天庭的"知识桥梁"
- **闹闹的特殊之处**：他是第一个"用科学知识而非神力"完成这个使命的后人——这意味着天道规则已经开始偏向科学侧
- **揭示时机**：第80章左右，太白金星在一次深夜对话中透露

### 4.4 暗线三：封印真相

- **混元锁界大阵**：一个上古封印，隔绝了天庭所在的空间与"天外天"（更高维度的位面）
- **封印的本质**：不是"关住什么东西"，而是"保护"——天外天有一种"规则侵蚀"现象，会改写低维世界的物理规则
- **神力是封印的钥匙**：以前纯神力驱动封印→神力被规则侵蚀→封印变弱
- **科学是新钥匙**：科学是对物理规则的"描述"而非"改写"→用科学加固封印→封印不再受规则侵蚀
- **终局**：第200章，闹闹带领神仙们用"神力+科学"双模式重建了封印，天庭进入"科学辅助神力"的新纪元
