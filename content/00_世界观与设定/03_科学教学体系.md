## 一、总纲

### 1.1 卷序设计逻辑

```
知识视野从"自己"出发，逐步放大——皮亚杰认知发展规律↓

第一卷（1-50）  人体/生活篇  🚽    "我的身体"      ← 最近
第二卷（51-100） 物理篇       ⚡    "我身边的世界"   ← 近
第三卷（101-150）化学+生物篇   🧪    "我看不见的世界" ← 远
第四卷（151-200）天文地理+工程  🌍    "我头顶的宇宙"   ← 最远
```

**开篇策略**：屎尿屁是3-10岁儿童的"注意力核武器"——放屁、鼻屎、大便、打嗝、尿尿……每个话题都自带笑声。先用50章人体篇建立信任和追更习惯，再逐步引入"正经"科学领域。

### 1.2 教学理念

```
闹闹的教学三原则：

1. 不问不讲 → 知识必须来自"翻车现场"的自然好奇
   不是"今天我们来学放屁"，而是"悟空你用金箍棒捅肚子干什么？！"

2. 不动不学 → 每个知识必须配一个实验
   不是"消化产生气体"，而是"八戒吃了三斤豆子然后发生了什么"

3. 不笑不算 → 实验成功或失败都要好玩
   不是"这个实验证明了……"，而是"哈哈哈哈哪吒你又炸了"+"但你学会了吗"
```

## 三、单章写作模板（课堂实验体·2300字）

```
每章2300字 ≈ 13分钟音频

【引入·翻车触发 · 200字】
  闹闹走进实验室 / 某个神仙闯了祸
  不是"今天学XX"，是"哪吒你又把什么炸了？！"
  参考《米吴科学漫画》：知识来自翻车现场

【课堂·知识讲解 · 500字】
  闹闹用孩子的理解方式讲解科学原理
  2-3个关键概念，不超3句核心解释
  穿插神仙的离谱理解（笑点）
  闹闹的"其实呢……"开场

【实验·翻车演示 · 700字】
  神仙用神力做实验 → 必然翻车
  翻车不是失败，是"用另一种方式证明科学是对的"
  描写重点：翻车的过程 + 围观神仙的反应
  每个神仙的翻车风格不同（哪吒=砰/悟空=轰隆/嫦娥=美但歪/八戒=吃）

【爆笑高潮·围观吐槽 · 400字】
  翻车后连锁反应 + 围观神仙的神吐槽
  至少3个不同神仙的反应（信服的/质疑的/嘲笑别人的）
  哪吒和二郎神互怼是固定节目
  太白金星记错误笔记是固定笑点

【知识总结·一语道破 · 200字】
  闹闹收尾，用"人话"+"神话"总结知识点
  慢速，给读者消化时间

【彩蛋钩子·下集预告 · 300字】
  某神仙留下"作业"（必然做错）
  或下一位神仙的恐惧表情（看到闹闹的实验计划表）
  收尾留钩："闹闹老师……下节课真的是那个实验吗？" 闹闹："对。"
  给读者翻下一页的理由

【🔬 闹闹的科学小实验 · 附页】
  （不在正文2300字内，作为每章结尾彩蛋板块）
  对标《科学实验王》《儿童科学百问百答》
  材料/步骤/现象/原理/安全提示
  3分钟能完成，家里材料
```

## 四、翻车触发公式

```
【课堂翻车公式（全书通用）】

闹闹引出【某个科学问题】（通常来自翻车现场）
  → 某位神仙自信满满："这有什么难的，看我的！"
  → 用神力做实验（错误方法）
  → 翻车（笑点）
  → 围观神仙吐槽（笑点叠加）
  → 闹闹解释真正的科学原理
  → 另一位神仙用正确方法重做（或验证）
  → 成功（爽点+知识点落地）
  → 翻车神仙："……好吧。科学是对的。"
  → 闹闹："这就是——"→全班齐声："——科学！"

示例（第1章·消化系统产气）：

  悟空：（自信）"人为什么会放屁？简单！肚子里有气，放出来就是屁！"
  闹闹："对。那气从哪来？"
  悟空："…………吃进去的？"
  闹闹："不是。是你肚子里的细菌分解食物产生的。"
  悟空："？？？俺老孙肚子里有细菌在吃俺老孙的东西然后放屁？？？"
  哪吒："哈哈哈哈哈哈——"
  悟空为了验证，吃了一整筐豆子。
  一个时辰后。
  悟空："闹闹老师……你说得对……太多了……控制不住……"
  实验室全员撤离。
  闹闹在走廊里继续讲："这就是肠道菌群。它们帮你消化，副产品是气体。"
  哪吒（捂着鼻子）："所以是细菌放的屁？"
  闹闹："对。你刚才的笑声是五十步笑百步。"
  太白金星（疯狂记笔记）："细菌……会放屁……记下了。"
  闹闹看了一眼笔记本："金星爷爷，你没记错但你不能只记'细菌会放屁'……"
```

## 五、每章"家庭小实验"设计原则

1. **材料家家有**：纸杯/吸管/橡皮筋/柠檬/醋/小苏打/气球/磁铁/手电筒/镜子
2. **3分钟能做完**：不超过5步
3. **结果直观可见**：孩子能立刻看到现象
4. **安全第一**：绝不用火/刀/电（如需用打火机，注明"请大人帮忙"）
5. **跟章节知识点一一对应**：实验=本章知识的动手版
