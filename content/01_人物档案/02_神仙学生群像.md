## 01 · 哪吒

### 基础档案

| 项目 | 内容 |
|------|------|
| 全称 | 哪吒三太子 |
| 神龄 | 三千多岁（外表永远12岁） |
| 天科办身份 | 最让人头疼的学生 + 实验品消耗量纪录保持者 |
| 每周炸实验室次数 | 平均2.3次（正在进步，第一个月是平均4.7次） |
| 口头禅 | "我觉得不对。" / "看我的！" / "……好吧，你是对的。" |
| 最大恐惧 | 闹闹说"哪吒同学，请你到办公室来一趟" |

### 性格画像

- **核心性格**：自信过剩 + 嘴硬 + 但输了会认（这是他的可爱之处——炸完实验室会低头小声说"好吧我错了"）
- **学习风格**：动手型学习者（"不动手我不信"→动手就炸→炸了才学）
- **笑点锚定**：全书最稳定的笑点引擎——"哪吒+任何实验器材=110%的翻车概率"

### 标志性翻车记录（前30章）

| 章 | 实验 | 翻车方式 | 哪吒的反应 |
|----|------|----------|-----------|
| 1 | 永动机验证 | 风火轮上转晕栽进桂花树 | "……这个不算！" |
| 4 | 弹性实验 | 把金箍棒当弹簧→弹飞撞南天门 | "……金箍棒先动手的" |
| 7 | 反作用力 | 风火轮火力全开→把实验室墙喷了一个洞 | "……我证明了反作用力！对吧闹闹？" 闹闹："对，但你也要赔这堵墙。" |
| 13 | 音调实验 | 用风火轮加热试管→试管炸了→碎玻璃飞到凌霄殿 | 玉帝："刚才有什么东西飞过去了？" |
| 21 | 静电实验 | 摩擦风火轮产生静电→碰谁谁炸毛→全实验室神仙的头发都竖着 | 哪吒："这个发型还挺酷的……" 悟空："我的毛！！" |

### 成长弧线

- 第1-30章：从"我不信"到"好吧我信了但我还是不服"
- 第31-60章：从"被科学打脸"到"主动用科学思维质疑"（"闹闹你说这个是化学变化，那证据呢？" → 闹闹内心OS：他终于开始用科学方法了）
- 第61-100章：从一个炸实验室的人变成"实验室安全守则制定者"（"因为我炸了太多次，我知道怎么防止炸。"）
- 第101-200章：天庭科学实验安全规范的总负责人，全书最不可能的职业转折

## 02 · 孙悟空

### 基础档案

| 项目 | 内容 |
|------|------|
| 全称 | 齐天大圣 / 斗战胜佛 |
| 神龄 | 不可考（石头里蹦出来的） |
| 天科办身份 | 最聪明但最不听讲的学生 + 动手能力天花板（翻车也是天花板级别） |
| 特殊技能 | 一个跟头十万八千里 + 七十二变（用来做科学实验时毁天灭地） |
| 口头禅 | "这个简单，俺老孙试一下！" / "闹闹老师，这个俺老孙能变一个出来吗？" |
| 最大矛盾 | 他是全班智商最高的学生，但注意力集中时间约等于一只真的猴子（约15秒） |

### 性格画像

- **核心性格**：天才型学渣——能瞬间理解复杂概念，但无法忍受超过两分钟的讲解（"闹闹老师你讲得太慢了俺老孙直接试一下"→试了就翻车）
- **学习风格**：直觉型+模仿型——不看说明书，直接上手。他能完美复制闹闹的实验操作，但永远不记得为什么
- **笑点锚定**：悟空式翻车 = 蛮力级翻车。哪吒的翻车是"砰"，悟空的翻车是"轰隆"——规模不在一个量级

### 标志性名场面

- **第2章·筋斗云超音速实验**：闹闹说"音速是340m/s"，悟空说"俺老孙一个跟头十万八千里，肯定超过音速了"→闹闹让他在月宫外面翻一个→因为没有空气，筋斗云失效了→悟空在月球轨道上飘了三个时辰才被嫦娥救回来→"闹闹老师，科学……有点东西。"
- **第8章·金箍棒平衡实验**：闹闹讲重心→悟空表演金箍棒顶盘子→金箍棒越变越长→盘子飞到凌霄殿（玉帝："今天已经是第二次有东西飞进来了。"）
- **第26章·七十二变与物质状态**：闹闹讲固液气三态→悟空说"俺老孙能变水蒸气"→变成了水蒸气→飘走了→回不来了（"闹闹老师！俺老孙怎么变回来？！" 闹闹："……你不是说你什么都能变吗？"）

### 成长弧线

- 第1-60章：从"俺老孙试一下"的无脑行动派 → 学会在动手前等三秒钟
- 第61-120章：展现出惊人的科学直觉（能猜对结果但说不出为什么）→ 闹闹帮他建立"从直觉到理论"的思维路径
- 第121-200章：成为闹闹最得力的实验搭档——"悟空你来做演示，大家看到没？这就是——别！别翻跟头！好，演示结束。"

### 音频角色设定

| 项目 | 内容 |
|------|------|
| 音色 | `zh_male_sunwukong_uranus_bigtts` — **猴哥 2.0** |
| 选角理由 | 这是火山引擎**专门为孙悟空定制的音色**！辨识度拉满，孩子一听就知道是孙悟空。不需要"像孙悟空"，它就是孙悟空 |
| 语速 | 快+跳跃感（猴子的语言节奏） |
| 语调 | 上扬+起伏大，带着一种"天不怕地不怕"的劲头 |
| 情绪范围 | 兴奋/跃跃欲试（35%）→ 翻车后的憨笑/挠头（25%）→ 认真思考的罕见时刻（15%）→ "闹闹老师俺老孙服了"的真诚（25%） |
| 独家音色特征 | "俺老孙"三字必须带猴子特有的颗粒感。笑声要爽朗（"哈哈哈哈"不是"嘿嘿嘿"） |

## 03 · 嫦娥

### 基础档案

| 项目 | 内容 |
|------|------|
| 全称 | 广寒仙子 |
| 神龄 | 四千多岁（外表约25岁） |
| 天科办身份 | 全班第一（四连冠"天庭科学家流动红旗"）+ 闹闹在天庭的"临时妈妈" |
| 特殊技能 | 对天文知识的天然直觉（因为住在月球上）+ 全班最认真的笔记（太白金星抄她的笔记） |
| 口头禅 | "闹闹老师，为什么……？" / "原来是这样……好美。" |
| 最大特点 | 她是全班唯一一个觉得"科学好美"而不是"科学好难"的学生 |

### 性格画像

- **核心性格**：文艺少女型学霸——不跟人争第一，但每次考试稳稳第一
- **学习风格**：感性驱动型——不是因为"这个知识有用"，而是因为"这个知识好美"。她学月相变化是因为"我想知道每天晚上我看到的那个蓝色星球（地球）是什么样子的"
- **与众不同之处**：其他神仙把科学当工具，嫦娥把科学当诗

### 标志性名场面

- **第3章·月相实验**：闹闹用月饼+手电筒演示月相变化→嫦娥："我等了三千年终于知道为什么月宫的门有时候会被地球照亮了……科学好美。"
- **第14章·月球生态**：嫦娥用玉兔的围栏建了月球的第一个"生态缸"→里面的桂花苗活了→嫦娥哭了（"这是我第一次在月球上种活东西"）→哪吒在旁边小声说"你哭什么啊这有什么好哭的"→闹闹瞪了他一眼→"哪吒同学，你不懂。"
- **第55章·光合作用**：闹闹讲光合作用→嫦娥举手："所以桂花树的每片叶子都是一个小小的厨房？"→闹闹："对！用阳光做菜！"→嫦娥眼睛里亮了星星

### 成长弧线

- 第1-50章：从天庭文艺委员→"天庭第一位生物学家"
- 第51-100章：月宫生态实验室正式挂牌→嫦娥成为天庭科学实验的标志性人物
- 第101-200章：嫦娥主导的"月宫农场"成为天庭与凡间科学交流的窗口

### 音频角色设定

| 项目 | 内容 |
|------|------|
| 音色 | `zh_female_roumeinvyou_uranus_bigtts` — **柔美女友 2.0** |
| 选角理由 | 温柔+优雅+知性的女声，完美匹配嫦娥"月上仙子"的气质。带一点空灵感，但不疏离——嫦娥在闹闹面前是亲切的"临时妈妈"，不是高冷女神 |
| 语速 | 中等偏慢（嫦娥说话不紧不慢，每个字都像月光落在地上） |

| 独家音色特征 | "好美"二字是嫦娥的标志性感叹——每次说完这两个字要有一个微小的静默（她在消化被科学震撼的情绪）。问"为什么"时尾音微微上扬，跟哪吒的"为什么"（质疑式）要有明显区分 |

## 04 · 太白金星

### 基础档案

| 项目 | 内容 |
|------|------|
| 全称 | 太白金星 |
| 神龄 | 不可考（他自己说"比玉帝年轻一点点"） |
| 天科办身份 | 班主任/教务主任/和事佬/擦屁股专员/天科办主任 |
| 特殊技能 | 在天庭所有部门之间协调（"协调"=替哪吒赔被炸坏的实验室器材） |
| 口头禅 | "林顾问说得对。" / "大家冷静……" / "这个账能不能报销？" |
| 最大特点 | 他是天庭唯一一个认真记笔记的神仙——但笔记全是错的 |

### 性格画像

- **核心性格**：天庭第一老实人。永远在替别人收拾烂摊子，永远不生气，永远笑眯眯
- **学习风格**：死记硬背型——拼命记笔记但理解能力有限，笔记内容经常出现惊天错误（"金星爷爷，你写的这个'光合作用=植物吃阳光拉氧气'……虽然方向对了但你不能写'拉'啊"）
- **笑点锚定**：太白金星的笔记是全书最稳定的隐性笑点——每次闹闹检查他的笔记时都会出现一段"纠错"环节，错误内容一次比一次离谱

### 标志性名场面

- **第10章·笔记检查**：闹闹检查太白金星的笔记——"惯性的定义：物体保持摔跤的倾向"→闹闹沉默了三秒→"金星爷爷，这是哪吒摔跤那天你记的，对吧。"→太白金星："对……那天太混乱了……"
- **第33章·替哪吒赔钱**：哪吒炸了实验室第12次→太白金星默默掏出功德卡→闹闹："金星爷爷，你不能每次都替他赔。"→太白金星："我知道……但我习惯了。"

### 成长弧线

- 始终是一个"靠谱的背景板"——他不需要巨大的成长，他是天庭的锚点人物
- 后期的变化：从"被动擦屁股"到"主动预防翻车"（第80章开始，他会在哪吒做实验前把所有易碎品搬走）

| 选角理由 | 温润如玉的老年声线，不急不躁，慈祥中带着一丝丝无奈（"又炸了啊……"）。悠悠君子=谦谦君子的老年版，完美匹配太白金星的老好人气质 |
