## 六、章节时长规格（有声书制作核心参数）

### 6.1 字数→时长换算

```
儿童有声书标准语速：160-180字/分钟
（比成人有声书慢10-15%，给孩子留理解时间）

每章2300字 ÷ 160字/分钟 ≈ 14.4分钟
每章2300字 ÷ 180字/分钟 ≈ 12.8分钟

→ 每章音频时长：10-15分钟 ✅
```

### 6.2 全系列时长预估

| 项目 | 数值 |
|------|------|
| 每章平均时长 | 13分钟 |
| 全系列200章 | 2,600分钟 = **43.3小时** |
| 每卷50章 | 650分钟 = **10.8小时/卷** |
| 适合收听场景 | 每天1章（睡前13分钟）/ 每天2章（上学路上+睡前） |

### 6.3 单章时间分配（约13分钟）

| 板块 | 字数 | 时长 | 说明 |
|------|------|------|------|
| 【引入】翻车触发 | ~200字 | ~1.2分钟 | 快节奏，笑声钩子 |
| 【课堂】知识讲解 | ~500字 | ~2.8分钟 | 闹闹主讲，中速 |
| 【实验】翻车/演示 | ~700字 | ~4.0分钟 | 全书最精彩段落，神仙群戏 |
| 【高潮】围观反应+神吐槽 | ~400字 | ~2.3分钟 | 笑点密集，节奏快 |
| 【知识总结】一语道破 | ~200字 | ~1.2分钟 | 闹闹收尾，慢速强化记忆 |
| 【彩蛋】下集钩子+家庭小实验 | ~300字 | ~1.5分钟 | 轻快，留钩 |

### 6.4 亲子收听节奏设计

```
前1分钟：抓住注意力（翻车/爆炸/神仙出糗）
  ↓
2-6分钟：知识核心（实验+讲解，节奏放缓）
  ↓
7-10分钟：笑点高潮（围观反应+神仙互怼）
  ↓
11-13分钟：收束+实验彩蛋（安静下来，准备入睡）
```

这个节奏专为**睡前收听**设计：开场炸一下→中间学知识→结尾笑完安静下来，孩子正好困了。

## 七、音频制作技术规范

### 7.1 空间声场分配

| 场景类型 | 混响类型 | 混响量 | 说明 |
|----------|----------|--------|------|
| 天庭室内（实验室/凌霄殿/丹房） | Plate/Chamber | 30-40% | 石墙大殿的自然回响 |
| 月宫 | Plate | 50-60% | 空旷大殿+严重回响 |
| 月面（宫外） | 无混响 | 0% | 真空=无空气=无声音传播（实际制作中完全无声不现实，用极干声代替） |
| 龙宫（水下） | Room（加低频EQ） | 20-30% | 被水压包裹的"闷"感 |
| 火焰山（户外） | 无混响+远处Delay | 0%（近处）/ 10%（远处） | 开阔山地 |
| 凡间学校 | 无混响 | 0% | 真实感 |
| 花果山（户外山林） | 自然回响 | 15-25% | 山谷回声 |
| 地府 | 长混响（Hall） | 60-70% | 阴间的空旷感 |

### 7.2 角色EQ建议

| 角色类型 | EQ建议 |
|----------|--------|
| 闹闹（童声） | 保留高频，不加额外处理（童声自带清晰度） |
| 哪吒/悟空（高频动作派） | 轻微提升2-5kHz（增加穿透力） |
| 嫦娥（温柔女声） | 轻微提升低频（增加温暖感） |
| 太白金星/土地公公（老年） | 轻微衰减高频（模拟年龄感） |
| 雷公（大嗓门） | 限制器保护（防爆音） |
| 旁白 | 保持中性，不加EQ |

### 7.3 特殊音效标记

| 音效 | 出现频率 | 制作建议 |
|------|----------|----------|
| 哪吒翻车爆炸 | 每2-3章 | 固定音效库，每次加微小变调以区分 |
| 悟空筋斗云 | 每5-10章 | "嗖——"声，带多普勒效应 |
| 八戒呼噜 | 每3-5章 | 固定音效（低沉的猪哼声） |
| 雷公打雷 | 雷公出场时 | 低频轰隆，注意不要压过人声 |
| 太白金星叹气 | 每2-3章 | 人声演绎（不靠音效） |
| 玉兔咀嚼 | 月宫场景 | 清脆的咀嚼声 |
| 实验室试管碰撞 | 实验室场景 | 玻璃碰撞声 |
| "闹闹的科学小实验"BGM | 每章结尾 | 固定短BGM（约15秒）作为板块标识 |

### 7.4 音量平衡

```
旁白基准音量：-3dB
角色对话：-3dB ~ -6dB（跟旁白同级别）
翻车音效：-6dB ~ -9dB（不能压过人声但要有冲击力）
环境底噪：-18dB ~ -24dB（能感知但不分散注意力）
BGM（背景）：-15dB ~ -20dB
BGM（实验板块标识）：-6dB
```

| 存档母带 | WAV | 48kHz | 24bit |
