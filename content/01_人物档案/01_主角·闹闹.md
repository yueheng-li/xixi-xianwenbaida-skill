## 性格画像

### 三层性格结构

```
外表层（神仙们看到的）：
  "这个小孩好说话，好欺负，好像什么都不懂"
  → 笑起来露出缺了一颗的门牙
  → 说话慢条斯理，喜欢用"其实呢……"开头

中间层（读者看到的）：
  "这个小孩是你们所有人的科学老师"
  → 翻开课本就能碾压三千年的神仙
  → 面对翻车现场异常淡定（"习惯了"）
  → 对科学知识的认真程度超过大多数成年人

深层（玉帝和太白金星看到的）：
  "这个小孩是天道选中的桥梁"
  → 天生对"规律"有直觉——不是天才，是观察力
  → 不问"为什么"不是不好奇，是在脑子里先想三轮才问
  → 有超乎年龄的耐心（"哪吒炸了多少次我都没放弃他"）
```

### 性格标签

| 标签 | 说明 | 表现 |
|------|------|------|
| 淡定 | 面对任何神仙翻车都面不改色 | "哦，又炸了。没事，我带了备用试管。" |
| 认真 | 科学知识绝不马虎 | "不是'差不多'，是9.8m/s²。9.7也不行。" |
| 护短 | 不许外人说他的神仙学生不好 | 二郎神："这群神仙也太笨了。"闹闹："他们不是笨，是还没学会。你再说一遍试试。" |
| 小大人 | 说话语气像个小老师 | "哪吒同学，请你回到座位上，实验还没结束。" |
| 悄悄想家 | 偶尔会想妈妈 | 第30章在天庭过夜，抱着哮天犬（"它暖暖的，像家里的狗"）——这个细节让嫦娥心疼了一整卷 |

### 内在矛盾

- **7岁的身体 × 30岁的表达能力**：他不是穿越者，但科学知识赋予了他超越年龄的表达能力。有时候会说出大人话，但偶尔还是会露出小孩的一面（看到蟠桃园的大桃子会嗷嗷叫、跟悟空打游戏机）
- **老师身份 × 小孩心态**：他教神仙的时候严肃认真，但教完之后会去爬花果山的树、跟小猴子们玩成一团

## 能力设定

### 在天庭的特殊权限

| 能力 | 说明 |
|------|------|
| 祥云专车 | 每周三下午，太白金星派专车接（凡间看不到） |
| 天界WiFi | 手机在天庭能用（雷公电母维护的服务器），信号时好时坏 |
| 课本传送 | 需要什么资料，在手机上下载→天庭AI自动翻译成"神仙版" |
| 科学免疫 | 在天庭时不受某些仙术影响（因为"科学不归仙术管"——后来成为剧情伏笔） |
| 天庭安全最高权限 | 玉帝特批：闹闹在天庭的安全优先级高于任何神仙 |

### 闹闹的"金手指"

闹闹没有超能力。他的"金手指"是三个：

1. **三年级科学课本**：在天庭这是唯一正确的科学文献。神仙们称之为"天书"（太白金星原话："这不是天书，是三年级教材。"）

2. **手机+天界AI翻译**：
   - 百度/科普网站上的知识 → 手机下载 → 天界AI翻译成"神仙版"
   - 翻译规则：DNA="生命的天书符咒"，光合作用="草木吸取日月精华的正确姿势"，万有引力="天道互吸律"
   - 闹闹每次用手机查资料，雷公就得跑过去调天线（"信号又没了！太白金星！那个小孩又在搜东西！"）

3. **无敌的耐心**：
   - 哪吒炸了七次实验室，他准备了八套备用器材
   - 悟空一个问题问二十遍，他回答了二十一遍
   - 八戒睡着了，他在八戒的呼噜声中继续讲课（"猪八戒同学，你醒着的时候也在吸收知识——通过空气振动，这个叫声音的传播。"）

## 音频角色设定

| 项目 | 内容 |
|------|------|
| 音色 | `zh_male_tiancaitongsheng_uranus_bigtts` — **天才童声 2.0** |
| 选角理由 | 天然童声 + "天才"气质（聪明/认真的小孩感），完美匹配闹闹的"小大人"性格 |
| 语速 | 中等偏慢（闹闹说话慢条斯理，"其实呢……"需要给听众留反应时间） |
| 语调 | 平稳为主，翻车时带一丝无奈的叹气感，讲科学知识时自动切换到"小老师"模式（认真但不严肃） |
| 情绪范围 | 淡定（50%）→ 认真讲解（30%）→ 无奈叹气/扶额（15%）→ 开心的笑（5%，爬树/吃蟠桃/做实验成功时） |
| 备用方案 | `zh_male_naiqimengwa_uranus_bigtts` — 奶气萌娃 2.0（如果天才童声听起来太"大"，可以换成更幼态的奶气萌娃） |
