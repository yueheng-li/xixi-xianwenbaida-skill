# 06-儿童内容智能优化规范

## 优化目标

对成品视频从科学准确性、儿童安全、趣味密度、传播力四个维度全面审核。

---

## 五大优化维度

### 1. 科学准确性审查（最重要）

**检查项**：
- [ ] 知识点表述正确（参考源：小学科学课标）
- [ ] 类比恰当（"硫化氢就是臭鸡蛋味"=正确 / 不能误导）
- [ ] 没有以偏概全
- [ ] 数字准确
- [ ] 神仙用神力验证的过程没有歪曲科学原理

**错误修正示例**：

| 错误 | 问题 | 修正 |
|------|------|------|
| "屁里面有甲烷所以会臭" | 甲烷无味，臭味来自硫化氢 | "屁里的硫化氢——就是臭鸡蛋的味道" |
| "冰比水轻所以浮在水上" | 太简略 | "水结冰后体积变大、密度变小" |

### 2. 儿童趣味性审查

- [ ] 笑点密度：每15秒至少一次刺激（笑点/拟声词/翻车）
- [ ] 对话像真小孩和真神仙在说话，不像科普文章
- [ ] 翻车场面够壮观
- [ ] 拟声词都有对应音效
- [ ] 闹闹的淡定和神仙的炸毛形成反差

### 3. 儿童安全审查（关键）

| 风险类型 | 排查内容 | 处理 |
|----------|----------|------|
| 火的实验 | 哪吒用风火轮加热 | 闹闹戴护目镜，标注"请勿模仿" |
| 化学药品 | 不可食用物质 | 家庭实验只用厨房材料 |
| 电的实验 | 闪电/静电 | 限制在"气球摩擦头发"级别 |
| 吞咽风险 | 小物件 | 不出现可吞咽材料 |
| 憋气/窒息 | 屏住呼吸的行为 | **绝对禁止** |

### 4. 家长友好度审查

- [ ] 是否有教育价值
- [ ] 屎尿屁尺度是否适度（好笑不下流）
- [ ] 神仙是否尊重闹闹老师（翻车后认错）
- [ ] 语言是否文明
- [ ] 价值观是否正向（科学精神、诚实认错、好奇探索）

### 5. 传播力评估

| 维度 | 评估标准 | 目标分 |
|------|----------|--------|
| 完播率 | 开头抓人+全程有趣 | ≥8 |
| 点赞 | 好笑或"哇" | ≥6 |
| 收藏 | 家长觉得"值得收藏给孩子看" | ≥7 |
| 评论 | 结尾有互动引导 | ≥7 |
| 转发 | 家长想转发给其他家长 | ≥6 |
| 关注转化 | 角色可爱度+知识期待 | ≥7 |

---

## 儿童专项避雷清单

### 绝对禁止
- 真实爆炸/火灾/血腥/伤害/死亡画面
- 恐怖/惊吓元素
- 可模仿的危险操作
- 成人化语言（网络梗/成人笑话）
- 政治/宗教内容

### 需谨慎
- 屎尿屁：每集不超过2处
- 神仙翻车：搞笑但不羞辱，翻车后必须有"认错→学习"弧
- 科学术语：翻译成儿童语言（"硫化氢=臭鸡蛋味"）
- 片尾建议加"看完去做实验吧！"鼓励离开屏幕
