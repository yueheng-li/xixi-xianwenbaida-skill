# 01-儿童科学内容文案规范（视频+音频）

## IP核心

《天庭幼儿园：今天神仙老师教什么》—— 7岁科学课代表闹闹是天庭最小学术权威。

**人设锚点**：
- **闹闹**：7岁，淡定认真，口头禅"这个很简单，我教你"，面对翻车面不改色
- **哪吒**：外表12岁，自信过剩，炸实验室纪录保持者，翻车后嘴硬"……这个不算！"
- **孙悟空**：天才型学渣，智商高但注意力≈15秒，"闹闹老师俺老孙直接试一下！"→必翻车
- **嫦娥**：温柔端庄，偶尔腹黑吐槽，广寒宫冷冻实验室负责人
- **八戒**：又懒又馋，偶尔歪打正着

**核心风格**：风趣幽默，闹闹淡定认真 vs 神仙咋咋呼呼，反差萌是核心笑点。孩子一听就爱上。

---

## 类型选择

| 类型 | 时长 | 字数 | 标志 |
|------|------|------|------|
| 🎬 视频 | 30-50秒 | 150-300字 | 对话体为主 |
| 🎧 音频 | 10-15分钟 | 1800-2500字 | 对话+旁白+丰富音效描写 |

**铁律**：如果用户没说视频还是音频，必须问。

---

## 🎬 视频文案（30-50秒）

### 三段式结构

**开篇（前3秒/15-25字）**：让小朋友停下来。

| 类型 | 示例 |
|------|------|
| 翻车开场 | 哪吒看着冒黑烟的风火轮："……闹闹老师，我不是故意的。" 闹闹淡定看了他一眼："你是第七次了。" |
| 问题开场 | 闹闹举着一个鸡蛋："同学们，为什么鸡蛋一煮就变硬，但鼻涕一擦就变软？" |
| 神仙提问 | 孙悟空举手："闹闹老师！为什么俺老孙放屁会响，但嫦娥仙子放屁——" 嫦娥："悟空。" |

**中段（4-35秒/80-200字）**：闹闹讲科学 + 神仙质疑/翻车。

标准节奏：闹闹讲解（1-2句）→ 神仙质疑（1句）→ 闹闹淡定回应（1句）→ 神仙动手→翻车（"砰！"）→ 闹闹平静总结（1句）

**结尾（5-8秒/20-35字）**：神仙懂了（或假装懂了）+ 家庭小实验彩蛋。

### 文案格式规范

**段名**：用描述性标题，不写固定时间码（时间由分镜环节自动计算）：
```
【开场】【悟空的蛤蟆理论】【闹闹的细菌课堂】【悟空炸毛】【收尾】
```

**角色标注**：每一句台词必须标注角色名，旁白用于场景叙事和段落衔接：
```
旁白：
"闹闹把课本往讲台上一放。啪一声，全场安静。"

闹闹：
"好。今天讲——人为什么会放屁。"

孙悟空：
"这个俺老孙懂！肚子里住了只蛤蟆——蛤蟆一叫，咕——气就顶出来了！"
```

**首集规则**：第一集开场旁白必须交代闹闹背景（七岁/科学课代表/太白金星招来的），后续集数直接进故事。

## 🎧 音频文案（10-15分钟）

### 开篇第一句（铁律）

> **"哈喽小伙伴们！欢迎收听月恒的有声书。今天聊——xxx。"**

这个开头是音频版的品牌标识，一字不能改。后面的内容可以自由发挥，但第一句必须是这句话。

### 音频完整结构

```
【开场白】30秒
哈喽小伙伴们！欢迎收听月恒的有声书。今天聊——[知识点]。
闹闹老师今天又要教哪位神仙做实验了呢？

【引入场景】2-3分钟
天庭日常场景→闹闹发现问题→神仙互动→引出科学问题
写法：对话体+适量旁白描绘场景（音频需要"让听众看见"）

【科学讲解】4-6分钟
闹闹用类比+举例讲科学原理→神仙插嘴/质疑/翻车
每2-3分钟一个小高潮（翻车/搞笑/音效）
写法：对话体为主，拟声词比视频多50%

【翻车高潮】2-3分钟
神仙大规模验证→天庭级别翻车
从"砰"升级为"轰隆"→拟声词全开+角色惊呼+围观吐槽
这是音频的"爽点峰值"

【知识总结】1-2分钟
闹闹淡定总结→神仙终于懂了→下一个知识的钩子
"明天闹闹老师要讲——"

【结束语】30秒
"今天的仙问百答就到这里啦！小朋友们记得做家庭小实验——[实验内容]。
把你的实验结果发在评论区，闹闹老师会看哦！我们明天见——拜拜！"
```

### 音频专属写作规则

- 对话体为主，旁白比视频多（音频需要描绘场景让听众"看见"画面）
- 拟声词比视频多50%（纯音频完全靠声音制造画面感）
- 每3分钟至少一个笑点/翻车/音效刺激（维持注意力比视频更难）
- 翻车音效要做足：爆炸声+玻璃碎+惊呼+围观吐槽，一个翻车场景至少3-4种音效
- 闹闹的淡定在音频中更突出：周围炸成一片，她依然慢条斯理
- 角色切换时，用旁白或音效过渡（音频无法靠画面切换区分角色）
- 每段对话前加角色标识（"闹闹说：""哪吒不服："），帮助听众分辨谁在说话

---

## 通用语言风格

### 做
- 单句≤20字（儿童能跟上）
- 闹闹：慢条斯理，"其实呢……""你看啊……"
- 哪吒：又快又自信，"我觉得不对！""看我的！""……好吧。"
- 悟空：更快更躁，"俺老孙直接试一下！"
- 拟声词丰富："砰！""呲——""哗——""咕噜咕噜""咣当！"
- 风趣幽默，笑点来自神仙翻车而非闹闹嘲笑他们

### 不做
- 不用超过小学三年级的词汇
- 不让闹闹生气/崩溃（淡定是核心人设）
- 不让神仙太蠢（只是用不惯科学方法）
- 每段闹闹讲科学不超过2-3句话

## 知识点选题体系

| 卷 | 主题 | 示例知识点 |
|------|------|----------|
| 🚽 第一卷·人体/生活 | 屎尿屁+身体奥秘 | 为什么放屁会臭 / 鼻屎怎么来的 / 肚子为什么叫 |
| ⚡ 第二卷·物理 | 声光电力磁 | 为什么天是蓝的 / 气球摩擦头发为什么吸纸片 |
| 🧪 第三卷·化学生物 | 物质变化+动植物 | 为什么洋葱辣眼睛 / 猫从高处跳下来为什么不受伤 |
| 🌍 第四卷·天文地理 | 太阳系+地球 | 为什么有白天黑夜 / 海水为什么是咸的 |

## 审核循环

文案生成后**必须停下来**等待用户审核：

```
生成文案 → 展示 → "请审核文案。通过请回复'通过'或'继续'，有修改意见请直接说。"
→ 等待反馈 → 通过→进入下一步 / 不通过→按意见修改→再次展示→循环
```
