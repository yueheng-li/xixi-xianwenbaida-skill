### 发布检查清单

```markdown
## 发布检查清单

- [ ] 音频已试听 / 视频已预览，无断句错误/角色错位
- [ ] 音量正常（-16 LUFS左右），无爆音
- [ ] 片头0.5s淡入，片尾1s淡出
- [ ] 段间停顿自然，角色切换流畅
- [ ] 角色声线无串音（同一角色全程同一音色）
- [ ] 标题含角色名+知识点
- [ ] 描述含具体细节（不泛泛）
- [ ] 标签覆盖平台主要流量入口
- [ ] 科学知识点核查无误
- [ ] 儿童安全审查通过
- [ ] 家庭小实验材料安全
- [ ] 上传平台 → 验证 → 发布
```

## 标题公式

### 抖音视频标题（15-30字）

| 类型 | 公式 | 示例 |
|------|------|------|
| 悬念型 | `[知识点]居然[反常识]？[N]秒看完就懂了` | 放屁会臭居然是因为臭鸡蛋？45秒看完就懂了 |
| 翻车型 | `[神仙]又翻车了！[干了什么]—闹闹：[淡定吐槽]` | 哪吒又炸了！这次是因为在实验室里放屁——闹闹：第七版守则新加了这条 |
| 知识型 | `儿童科学：[一句话知识点]#仙问百答` | 儿童科学：为什么肚子会咕噜咕噜叫？#仙问百答 |

| 类型 | 公式 | 示例 |
|------|------|------|

## 分平台发布流程

### 🎬 抖音

```
1. 从归档目录 抖音/{卷}·{知识点}/ 取成品MP4
2. 复制发布信息.md中的标题+描述+标签
3. 打开抖音创作者中心 → 上传视频
4. 设置封面（建议选翻车瞬间或闹闹淡定脸）
5. 填写标题、添加话题标签
6. 定时发布或立即发布
7. 发布后24h回填数据到 07_发布记录.md
```

### 🎧 喜马拉雅

```
1. 从归档目录 喜马拉雅/{卷}·{知识点}/ 取成品MP3
2. 运行 ximalaya_upload.py（API方式）
   或手动上传到 studio.ximalaya.com
3. 选择对应专辑 → 上传音频
4. 填写标题、描述、标签
5. 发布
6. 发布后回填数据
```

## 批量发布

量产时统一生成发布包：

```
1. 遍历 抖音/ 目录下所有待发布视频 → 批量生成发布信息.md
2. 遍历 喜马拉雅/ 目录下所有待发布音频 → 批量生成发布信息.md
3. 统一审核 → 排期 → 批量上传或手动发布
```
